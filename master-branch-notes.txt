* Branches and release scheme

The release convention was discussed today between Tom Cobb and Ronaldo Mercado.

The first release from this "master" branch was release 5-6-6.

It is intended that further rhel7 releases will be 5-7 and higher

"master" branch is for rhel7 (EPICS R3.14.12.7)
"rhel6" branch is for rhel6 (EPICS R3.14.12.3)

Releases from "rhel6" branch should be 5-6-9 or higher

However there are RHEL7 releases 5-6-6 and 5-6-8 that were created
before this note was written to document the convention.
